# 🍕 مطعم بيتزا إيطاليا - مشروع تخرج

مشروع واجهة أمامية احترافي لمطعم بيتزا إيطالي، مبني باستخدام HTML5, CSS3, Bootstrap 5, JavaScript, jQuery, Sass, و Webpack.

## 🚀 التقنيات المستخدمة

- **HTML5** - هيكل الصفحات
- **CSS3 / Sass (SCSS)** - التنسيقات المتقدمة
- **Bootstrap 5** - الإطار التصميمي المتجاوب
- **JavaScript (Vanilla)** - التفاعلية
- **jQuery** - تسهيل التعامل مع DOM
- **Webpack** - تجميع وتحزيم الملفات
- **Git** - إدارة النسخ
